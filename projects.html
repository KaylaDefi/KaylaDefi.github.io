PacMen Exercise

This code is for a web-based game or simulation that creates and animates Pac-Man characters. When a new Pac-Man is made using makePac(), it is given a random position and velocity, and an image element is created and placed in the game area. The update() function moves each Pac-Man based on its velocity and checks for collisions with the window boundaries, reversing direction and cycling through images to simulate movement. The startGame() function sets this update process to repeat at intervals, creating the animation loop. Event listeners are added to DOM elements to create new Pac-Men and start the game when the user interacts with the page.

Eye Exercise

This code snippet creates an eye-following effect where two elements with the class 'ball' will track the movement of the user's mouse cursor on the screen. As the mouse moves, it updates the position of each 'ball' to match the cursor's position in percentage terms relative to the window's width and height, creating the illusion that the 'balls' are looking at the cursor.

Real Time Bus Tracker

This code initializes and manages a customized Google Maps instance, primarily to track and display bus locations. The map is styled with a custom color scheme and centered at specific coordinates. The addMarkers() function, called at regular intervals, fetches real-time bus location data from the MBTA API and updates or adds markers on the map to represent the buses. These markers, distinguished by different colors based on the bus direction, move to reflect the buses' current positions. The map and marker functionalities are initialized when the window loads.
